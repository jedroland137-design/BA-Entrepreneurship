# BA in Entrepreneurship Study Vault

This repository stores my working materials for IU courses in the BA in Entrepreneurship project.

## Purpose
- Keep course books and references in one stable place
- Store typed study notes
- Store exam triggers and self-tests
- Prevent loss of files across chats and uploads

## Folder guide
- `references/` = source logs and access notes
- `managerial-economics/` = notes, exam triggers, self-tests
- `managerial-accounting/` = notes, exam triggers, self-tests
- `thesis/` = reserved for later

## Working rule
Notes are only useful if they are tested. Read → compress → self-test → move on.
